# СНТ Бухгалтерия

Приложение для бухгалтерского учёта садоводческого некоммерческого товарищества (СНТ).

## Стек технологий

- **Python 3.14** — язык программирования
- **CustomTkinter** — современный GUI-фреймворк
- **SQLAlchemy 2.0** — ORM для работы с базой данных
- **SQLite** — встроенная база данных
- **openpyxl** — импорт/экспорт Excel
- **fpdf2** — генерация PDF с поддержкой кириллицы
- **matplotlib** — графики и диаграммы
- **Pillow** — работа с изображениями

## Установка и запуск

```bash
pip install -r requirements.txt
python main.py
```

## Ключевые архитектурные решения

### 1. ttk.Treeview вместо CTkTable
Нативная производительность на 300+ строках, встроенная сортировка по клику на заголовок, цветовые теги для статусов (оплачено — зелёный, частично — жёлтый, долг — красный, архив — серый).

### 2. EventBus (pub/sub)
Связь между табами без жёстких зависимостей. События:
- `member_updated` — обновление списка участников
- `fee_paid` — запись оплаты
- `theme_changed` — переключение темы
- `database_changed` — смена базы данных
- `status_message` — обновление статусной строки

### 3. Decimal для денег
`Numeric(12,2)` в БД, `Decimal` в Python — никогда `float` для денежных сумм.

### 4. fpdf2 для PDF
Поддержка кириллицы через TTF-шрифты. Использует DejaVuSans или системный Arial (Windows) как fallback.

### 5. Переключение БД
`dispose engine → create new engine → create_all → publish database_changed` — все табы автоматически перезагружаются.

### 6. Пеня рассчитывается на лету
`(сумма_долга × дневная_ставка × дни_просрочки)`, snapshot сохраняется при записи оплаты.

## Функциональность

### Участники
- Добавление / редактирование / архивирование / восстановление
- Поиск по ФИО и номеру участка
- Фильтрация: все / активные / архивные
- Карточка участника: ФИО, участок, площадь, телефон, email, адрес, заметки

### Членские взносы
- Создание / редактирование / удаление периодов
- Автоматическая генерация начислений для всех активных участников
- Расчёт: `ставка_за_сотку × площадь_участка`
- Запись полной или частичной оплаты
- Автоматический расчёт пени за просрочку

### Целевые взносы
- Кампании двух типов: фиксированная сумма или за сотку
- Создание / редактирование / удаление кампаний
- Генерация начислений
- Трекинг оплат
- Прикрепление документов к кампании

### Электроэнергия
- Управление тарифами (руб/кВт·ч)
- Ввод показаний счётчиков с автоматическим переносом предыдущего показания
- Расчёт потребления и суммы к оплате
- Обнаружение аномально высокого потребления (>3x от среднего)
- Общий счётчик СНТ

### Заметки
- Создание / редактирование / удаление заметок
- Форматирование текста (жирный, курсив, подчёркивание)
- Закрепление (pin) важных заметок

### Аналитика (дашборд)
- Карточки сводки: участники, членские взносы, целевые взносы, электроэнергия
- Круговая диаграмма по статусам оплаты членских взносов
- Столбчатая диаграмма оплат по периодам
- Таблица должников с сортировкой по сумме долга

### Сервис
- **Импорт** участников из CSV (разделители `;` или `,`) и Excel (`.xlsx`)
  - Гибкое определение колонок (русские и английские названия)
  - Пропуск существующих участков
- **Экспорт** списка участников в Excel и PDF
- **Экспорт** отчёта по должникам в PDF
- Автоматический бэкап при запуске приложения
- Ручное создание и восстановление резервных копий
- Ротация: хранятся последние 5 копий
- Создание новой базы данных
- Импорт и экспорт файла базы данных
- Переключение между несколькими базами данных
- Все табы автоматически перезагружаются при смене БД

## Тема оформления

Поддерживаются две темы: **тёмная** и **светлая**. Переключение через тогл в панели инструментов.

Цветовые теги строк в таблицах:
- Зелёный — оплачено
- Жёлтый — частично оплачено
- Красный — не оплачено
- Серый — архивный участник
